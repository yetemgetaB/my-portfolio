# ✈️ Sky Code Nexus - Flight Control Dev Console

🚀 **An innovative portfolio website combining aviation themes with modern web development**

## 🎯 Project Overview

Sky Code Nexus is a unique portfolio experience designed as a flight control system interface. Instead of controlling aircraft, visitors navigate through my career data, projects, and skills in an immersive, interactive environment.

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: TailwindCSS + Radix UI Components
- **Build Tool**: Vite
- **Routing**: React Router DOM
- **State Management**: React Query (TanStack Query)
- **Testing**: Vitest + Playwright
- **Icons**: Lucide React

## 🎨 Features

### 📊 Main Portfolio (`/`)
- **Mission Control Panel** (Left): System status, toggle switches, objectives
- **Code Deck** (Center): File explorer, code viewer, terminal interface  
- **Telemetry Dashboard** (Right): Live stats, GitHub integration, radar display

### 🔧 Admin Dashboard (`/admin`)
- Secure admin access for content management
- Real-time portfolio editing
- GitHub repository management
- Theme customization
- Analytics dashboard

### 🚀 GitHub Integration
- Auto-fetch repositories and project data
- Display commit history and contributions
- Show release information and download stats
- Real-time repository health monitoring

## 🎮 Interactive Elements

- **Functional toggle switches** with sound effects
- **Live radar display** with centered sweep animation
- **Keyboard shortcuts** for power users
- **Responsive design** with mobile optimization
- **Micro-interactions** and smooth transitions

## 📱 Responsive Design

- Desktop: Full 3-panel flight control interface
- Tablet: Collapsible panels with swipe gestures
- Mobile: Single-column layout with touch controls

## 🛸 Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm run test
```

## 🌐 Deployment

Ready for deployment on Vercel, Netlify, or any static hosting platform.
